# MASTER PROMPT — Build "6O.CJFX Trading Edge"

Build a modern, professional, high-performance web application called **6O.CJFX Trading Edge**.

The purpose of this application is to act as my daily pre-trade checklist, trading discipline system, emotional tracker, and trade decision framework. The goal is to prevent impulsive trades and ensure that only high-quality setups are executed.

This is a personal trading tool inspired by professional prop-firm discipline and institutional trading processes.

---

## Tech Stack

Use:

* React
* TypeScript
* Tailwind CSS
* Framer Motion
* Recharts
* React Hooks
* Local Storage Persistence

Do NOT use:

* Firebase
* Supabase
* MongoDB
* SQL
* Authentication
* Backend Server

The entire application should work as a static website and be deployable to GitHub Pages.

# Daily Trading Session Logic

This application should be designed around a single trading day.

Requirements:

## Daily Persistence

If the user refreshes the page:

* Keep all data
* Keep all checkboxes
* Keep all notes
* Keep emotional tracking
* Keep score calculations

Store everything using Local Storage.

---

## Automatic Daily Reset

Store the current trading date.

Example:

{
date: "2026-06-22",
data: {...}
}

Every time the application loads:

Check if:

savedDate !== currentDate

If true:

Automatically clear today's checklist.

Display notification:

"New Trading Day Started. Previous Session Cleared."

---

## Manual Reset Button

Add a visible button in the top-right corner:

🔄 Reset Trading Session

When clicked:

Show confirmation modal:

"Are you sure you want to reset today's trading session?"

Buttons:

* Cancel
* Reset Session

Upon confirmation:

* Clear all checklist data
* Clear emotional tracker
* Clear notes
* Reset Trade Quality Score
* Reset verdict
